Create a modern, premium e-commerce homepage for an Indian grocery and trader brand called “Asian Cart”.

Use the uploaded Asian Cart logo exactly as provided. Do not redraw or modify the logo. Use the horizontal logo version in the desktop header and the stacked logo version where needed on mobile.

Brand personality:
Premium, reliable, modern Indian grocery store, clean marketplace, quality ingredients, wholesale-friendly but still welcoming for households.

Primary colors:
- Deep forest green: #003C32 or similar
- Bright lime green: #9AD62A
- Fresh green: #57BF3C
- White / off-white backgrounds
- Charcoal grey for text
- Soft warm beige backgrounds for grocery sections

Build a fully responsive React/Vite homepage using Tailwind CSS.
